# Teslo API

## Description

Teslo shop

### Installation

```bash
$ pnpm install
```

### Running the database

```bash
$ docker-compose up -d
```

### Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

### Execute seed

```bash
$ GET: http://localhost:3000/api/seed
```
